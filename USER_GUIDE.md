## Lab Test Reference Ranges

| Test | Normal Range |
|------|--------------|
| **Fasting Blood Sugar** | < 100 mg/dL |
| **HbA1c** | < 5.7% |
| **Total Cholesterol** | < 200 mg/dL |
| **LDL Cholesterol** | < 100 mg/dL (optimal) |
| **HDL Cholesterol** | > 40 mg/dL (Men), > 50 (Women) |
| **Triglycerides** | < 150 mg/dL |
| **Blood Pressure** | < 120/80 mmHg |
| **Creatinine** | 0.7-1.3 mg/dL |
| **GFR** | > 60 mL/min |

## Troubleshooting

### Water not logging?
- Check internet connection
- Ensure you're logged in to your account
- Refresh page and try again

### Exercise calories seem wrong?
- Calorie burn is estimated based on intensity
- Actual burn varies by person, weight, fitness level
- Consider wearing a fitness tracker for accuracy

### Lab results show as warning/critical?
- Compare to normal ranges in the reference table
- Consult with your doctor for interpretation
- Some variations are normal

### Health score not increasing?
- Log more exercises (need 3+ per week)
- Track daily water intake
- Record lab test results
- Check health dashboard for current score
